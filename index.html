Terms Of Use

With ASSESS, you can automatically generate tests from RDF knowledge bases. ASSESS is open-source. The code for the ASSESS server can be found at github.com/AKSW/SPARQL2NL. The code for the client is available at github.com/AKSW/ASSESS. The prototype is provided as-is, i.e., without any guarantee of any kind. Please feel free to use it at will. If you have any questions, please contact {buehmann/usbeck/ngonga}@informatik.uni-leipzig.de. Enjoy. The ASSESS Team
